# إصلاح مشكلة Prisma Client

## المشكلة
خطأ: `Unknown argument 'sizeGuide'` عند محاولة حفظ منتج

## السبب
Prisma Client لم يتم تحديثه بعد إضافة حقل `sizeGuide` في قاعدة البيانات.

## الحل السريع

### الخطوة 1: إيقاف الخادم
إذا كان الخادم يعمل، أوقفه (Ctrl+C في terminal الخادم)

### الخطوة 2: تحديث Prisma Client
افتح terminal جديد في مجلد المشروع وقم بتشغيل:

```bash
cd backend
npx prisma generate
```

### الخطوة 3: إعادة تشغيل الخادم
قم بتشغيل الخادم مرة أخرى

## أو استخدم الملف الجاهز
قم بتشغيل: `update-prisma.bat` (انقر نقراً مزدوجاً عليه)

---

بعد ذلك، ستتمكن من حفظ المنتجات مع حقل `sizeGuide` بدون مشاكل! ✅
